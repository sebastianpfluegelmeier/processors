  - DigitalClip
  - HardCurve
  - SineWave
  - SawWave
  - TriangleWave
  - PulseWave
- dynamics
- filters
- modulation
- oscillators
  - Sine
  - Saw
  - Square
  - Tri
- utils
  - Invert
  - Constant
  - Mul
